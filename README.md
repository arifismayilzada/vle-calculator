# VLE Calculator

## Description
This project calculates vapor-liquid equilibrium (VLE) for a binary system
using Raoult's Law and the Antoine equation.

## Theory
- Antoine Equation for saturation pressure
- Raoult's Law for phase equilibrium

## Features
- Bubble point pressure calculation
- Vapor phase composition
- Beginner-friendly structure
